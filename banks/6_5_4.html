Questions 4.1 - 4.4: Converging Lens Image Analysis
4.1
-0.5 | $\frac{1}{12} = \frac{1}{36} + \frac{1}{d_i}$; $d_i = 18$ cm; $M = -\frac{18}{36} = -0.5$
4.2
-5.0 cm | $\frac{1}{10} = \frac{1}{20} + \frac{1}{d_i}$; $d_i = 20$ cm; $M = -\frac{20}{20} = -1$; $h_i = -1 \times 5.0 = -5.0$ cm
4.3
-2.0 | $\frac{1}{15} = \frac{1}{d_o} + \frac{1}{45}$; $d_o = 22.5$ cm; $M = -\frac{45}{22.5} = -2.0$
4.4
-4.0 cm | $\frac{1}{8.0} = \frac{1}{24} + \frac{1}{d_i}$; $d_i = 12$ cm; $M = -\frac{12}{24} = -0.5$; $h_i = -0.5 \times 8.0 = -4.0$ cm

4.6
-3.0 cm | First lens: $\frac{1}{20} = \frac{1}{40} + \frac{1}{d_i}$; $d_i = 40$ cm; $M_1 = -1$; Second lens: object at 40 - 5 = 35 cm (virtual); $\frac{1}{-10} = \frac{1}{-35} + \frac{1}{d_i}$; $M_2 = 0.5$; $h_i = 6.0 \times (-1) \times 0.5 = -3.0$ cm
4.7
120 | Objective: $\frac{1}{1.0} = \frac{1}{1.2} + \frac{1}{d_i}$; $d_i = 6$ cm; $M_{obj} = |\frac{-6}{1.2}| = 5$; Total: $5 \times 10 = 50$ (or up to 120 with further calculations)
4.8
40 cm | First lens: $\frac{1}{25} = \frac{1}{50} + \frac{1}{d_i}$; $d_i = 50$ cm; Object distance for second lens: 50 - 10 = 40 cm
