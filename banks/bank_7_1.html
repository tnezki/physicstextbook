Coulomb’s Law Solutions
3.1
0.216 N
$F = k\frac{|q_1 q_2|}{r^2} = (8.99\times10^9)\frac{(3.0\times10^{-6})(2.0\times10^{-6})}{(0.50)^2} = 0.216\text{ N}$
3.2
0.354 m
$r = \sqrt{k\frac{|q_1 q_2|}{F}} = \sqrt{(8.99\times10^9)\frac{(4.0\times10^{-6})(1.5\times10^{-6})}{0.43}} = 0.354\text{ m}$
3.3
$2.0\times10^{-6}$ C
$q_2 = \frac{Fr^2}{k q_1} = \frac{(0.90)(0.30)^2}{(8.99\times10^9)(5.0\times10^{-6})} = 2.0\times10^{-6}\text{ C}$
3.4
$1.0\times10^{-6}$ C
$q = \sqrt{\frac{Fr^2}{k}} = \sqrt{\frac{(2.25)(0.20)^2}{8.99\times10^9}} = 1.0\times10^{-6}\text{ C}$
