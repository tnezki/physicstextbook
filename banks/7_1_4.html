Scenario 4: Identical Charges

Complete Answer Key

Scenario 1
4.1
0.674 N → 0.169 N
Convert: $6.0\,\mu C = 6.0\times10^{-6}$ C $F = k\frac{q_1q_2}{r^2} = (8.99\times10^9)\frac{(6.0\times10^{-6})(2.0\times10^{-6})}{0.40^2} = 0.674\text{ N}$ Distance doubled → force ÷ 4 $F_{new} = 0.674/4 = 0.169$ N
Scenario 2
4.2
$3.0\times10^{-6}$ C
$q_2 = \frac{Fr^2}{k q_1} = \frac{(1.20)(0.30)^2}{(8.99\times10^9)(4.0\times10^{-6})} = 3.0\times10^{-6}$ C
Scenario 3

0.424 m
$r = \sqrt{k\frac{q_1q_2}{F}} = \sqrt{(8.99\times10^9)\frac{(3.0\times10^{-6})(5.0\times10^{-6})}{0.75}} = 0.424$ m
