Formula 1: Lens Equation
Formula: $$\frac{1}{f} = \frac{1}{d_o} + \frac{1}{d_i}$$

Variables: $f$ = focal length (cm), $d_o$ = object distance (cm), $d_i$ = image distance (cm)

Answer: _____ cm
3.4

Answer: _____ cm
Formula 2: Magnification (Lenses)
Formula: $$M = -\frac{d_i}{d_o} = \frac{h_i}{h_o}$$

Variables: $M$ = magnification (unitless), $d_i$ = image distance (cm), $d_o$ = object distance (cm), $h_i$ = image height (cm), $h_o$ = object height (cm)

Variables: $f$ = focal length (cm), $R$ = radius of curvature (cm)

Variables: $h_i$ = image height (cm), $M$ = magnification (unitless), $h_o$ = object height (cm)
