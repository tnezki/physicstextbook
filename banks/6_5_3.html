Formula 1: Lens Equation
Formula: $$\frac{1}{f} = \frac{1}{d_o} + \frac{1}{d_i}$$

Variables: $f$ = focal length (cm), $d_o$ = object distance (cm), $d_i$ = image distance (cm)
3.1
A converging lens has a focal length of 15 cm. An object is placed 30 cm from the lens. Calculate the image distance.
Answer: _____ cm
3.2
Using the lens equation, if $f = 10$ cm and $d_o = 20$ cm, find the image distance.
Answer: _____ cm
3.3
A lens produces an image 40 cm away when the object is 30 cm from the lens. Calculate the focal length.
Answer: _____ cm
3.4
An object is placed 25 cm from a lens with focal length 10 cm. Determine the image distance.
Answer: _____ cm
Formula 2: Magnification (Lenses)
Formula: $$M = -\frac{d_i}{d_o} = \frac{h_i}{h_o}$$

Variables: $M$ = magnification (unitless), $d_i$ = image distance (cm), $d_o$ = object distance (cm), $h_i$ = image height (cm), $h_o$ = object height (cm)
3.5
An object 4.0 cm tall is placed 20 cm from a converging lens. The image forms 60 cm from the lens. Calculate the magnification.
Answer: _____ (unitless)
3.6
If the image distance is 50 cm and the object distance is 25 cm, what is the magnification?
Answer: _____ (unitless)
3.7
An object 6.0 cm tall produces an image that is 18 cm tall. Calculate the magnification.
Answer: _____ (unitless)
3.8
A lens produces a magnification of -2.0. If the object is 5.0 cm tall, what is the image height?
Answer: _____ cm
Formula 3: Focal Length and Radius of Curvature
Formula: $$f = \frac{R}{2}$$

Variables: $f$ = focal length (cm), $R$ = radius of curvature (cm)
3.9

Answer: _____ cm
3.10

Answer: _____ cm
3.12

Answer: _____ cm
Formula 4: Image Height Using Magnification
Formula: $$h_i = M \times h_o$$

Variables: $h_i$ = image height (cm), $M$ = magnification (unitless), $h_o$ = object height (cm)
3.13

3.14

Answer: _____ cm
3.15

Answer: _____ cm
3.16
An object 10 cm tall produces an image with magnification -0.5. Find the image height.
Answer: _____ cm
