Section 8.3: Series and Parallel Circuits

Bank 1: Vocabulary Assessment

Multiple Choice | Vocabulary Mastery

Series circuit (1.1 – 1.4)
A circuit in which components are connected one after another in a single path; current is same everywhere.
1.1
A series circuit is best described as:
A) A circuit with multiple paths
B) A circuit with one continuous path for current
C) A circuit with zero resistance
D) A circuit with constant voltage across all components
E) A circuit without a power source
1.2
In a series circuit, the current:
A) Is different in each component
B) Is zero everywhere
C) Is the same at all points
D) Doubles in each resistor
E) Flows only through the battery
1.3
If one component in a series circuit is removed, the circuit will:
A) Continue operating normally
B) Have increased current
C) Stop working entirely
D) Become parallel
E) Double its voltage
1.4
Adding another resistor in series will:
A) Decrease total resistance
B) Leave resistance unchanged
C) Increase total resistance
D) Eliminate voltage
E) Increase current
Parallel circuit (2.1 – 2.4)
A circuit in which components are connected along multiple paths; voltage is same across all branches.
2.1
A parallel circuit contains:
A) Only one path for current
B) Multiple paths for current
C) Zero voltage
D) Infinite resistance
E) No current flow
2.2
In a parallel circuit, the voltage across each branch:
A) Is different
B) Is zero
C) Is the same
D) Doubles in each branch
E) Depends only on resistance
2.3
If one branch of a parallel circuit is disconnected, the remaining branches:
A) Stop working
B) Continue operating
C) Have zero voltage
D) Reverse current
E) Lose all resistance
2.4
Adding an additional branch in parallel will:
A) Increase total resistance
B) Leave resistance unchanged
C) Decrease total resistance
D) Eliminate current
E) Increase voltage
