SECTION 1.5

Vector Addition

When multiple forces act on an object, or when motion involves multiple directions, we must add vectors together. In this final section of Module 1, you'll learn to add vectors both graphically and analytically, bringing together all the skills you've developed throughout this module.

Learning Objectives

  • I can add vectors graphically using the tip-to-tail (head-to-tail) method.
  • I can add vectors graphically using the parallelogram method.
  • I can add vectors analytically by adding their components.
  • I can calculate the magnitude and direction of the resultant vector and verify results using multiple methods.

Introduction

Imagine you're hiking through a forest. First, you walk 3 km east, then 4 km north. How far are you from your starting point, and in what direction? This is a vector addition problem! The answer isn't simply 3 + 4 = 7 km, because the directions matter.

In physics, we constantly need to add vectors: multiple forces acting on an object, an object moving in multiple directions, electric fields from several charges, and much more. In this section, you'll master both graphical methods (using drawings) and analytical methods (using math) to add vectors accurately.

The Resultant Vector

When we add two or more vectors, the result is called the resultant vector (or just the resultant). The resultant has the same effect as all the individual vectors combined.

For example, if two forces push on a box—one 30 N to the right and one 40 N upward—the resultant force is a single force that has the same effect as both forces acting together.

Resultant Vector: The vector sum of two or more vectors. It represents the combined effect of all the vectors.

Notation: If we add vectors $\mathbf{A}$ and $\mathbf{B}$, the resultant is $\mathbf{R} = \mathbf{A} + \mathbf{B}$

Graphical Method 1: Tip-to-Tail Method

The tip-to-tail method (also called head-to-tail method) is the most common graphical way to add vectors:

Steps for Tip-to-Tail Method:

  1. Draw the first vector to scale with correct direction (usually starting at the origin).
  2. Draw the second vector starting at the tip (head/arrowhead) of the first vector, maintaining its correct length and direction.
  3. Draw the resultant from the tail of the first vector to the tip of the last vector.
  4. Measure the resultant's length (gives magnitude) and angle (gives direction).

Key Point: Order Doesn't Matter

Vector addition is commutative: $\mathbf{A} + \mathbf{B} = \mathbf{B} + \mathbf{A}$. You can add vectors in any order and get the same resultant. Try drawing B first, then A—you'll get the same result!

Graphical Method 2: Parallelogram Method

The parallelogram method is another graphical technique, especially useful for adding just two vectors:

Steps for Parallelogram Method:

  1. Draw both vectors starting from the same point (tail to tail).
  2. Complete a parallelogram using these vectors as adjacent sides.
  3. Draw the resultant as the diagonal of the parallelogram from the common starting point.
  4. Measure the resultant's length and angle.

Both the tip-to-tail and parallelogram methods give the same resultant—they're just different ways to visualize the same addition.

Analytical Method: Component Addition

While graphical methods give you a visual understanding, the analytical method using components is more accurate and works for any number of vectors.

Steps for Analytical Vector Addition:

  1. Break each vector into components: Find $A_x$, $A_y$, $B_x$, $B_y$, etc.
  2. Add all x-components together: $R_x = A_x + B_x + C_x + ...$
  3. Add all y-components together: $R_y = A_y + B_y + C_y + ...$
  4. Find the resultant's magnitude: $R = \sqrt{R_x^2 + R_y^2}$
  5. Find the resultant's direction: $\theta = \tan^{-1}\left(\frac{R_y}{R_x}\right)$

Real-World Application: Navigation

Pilots and ship captains constantly use vector addition. Wind affects a plane's motion—if a plane flies east at 500 km/h but there's a 100 km/h wind from the south, the plane's actual velocity (relative to the ground) is the vector sum of these two velocities. GPS systems and autopilots perform these vector calculations continuously to keep aircraft on course.

Example Problems

Problem-Solving Strategy

  1. List the Givens: Write down all vectors with magnitudes, directions, and/or components.
  2. Find the Formula: Identify which formulas you'll need.
  3. Substitute: Plug values into formulas.
  4. Solve: Calculate step-by-step.
  5. Check: Verify units and reasonableness.

Example 1: Adding Vectors in Component Form

Problem: Vector A has components $A_x = 12$ m and $A_y = 5$ m. Vector B has components $B_x = 8$ m and $B_y = 15$ m. Find the x-component of the resultant vector $\mathbf{R} = \mathbf{A} + \mathbf{B}$.

Step 1 - List the Givens:

  • $A_x = 12$ m, $A_y = 5$ m
  • $B_x = 8$ m, $B_y = 15$ m
  • Find: $R_x$

Step 2 - Find the Formula:

$$R_x = A_x + B_x$$

Step 3 - Substitute:

$$R_x = 12 \text{ m} + 8 \text{ m}$$

Step 4 - Solve:

$$R_x = 20 \text{ m}$$

Answer: The x-component of the resultant is 20 m.

Example 2: Finding Resultant Magnitude from Components

Problem: The resultant of two vectors has components $R_x = 24$ N and $R_y = 18$ N. What is the magnitude of the resultant?

Step 1 - List the Givens:

  • $R_x = 24$ N
  • $R_y = 18$ N
  • Find: magnitude $R$

Step 2 - Find the Formula:

$$R = \sqrt{R_x^2 + R_y^2}$$

Step 3 - Substitute:

$$R = \sqrt{(24)^2 + (18)^2} = \sqrt{576 + 324}$$

Step 4 - Solve:

$$R = \sqrt{900} = 30 \text{ N}$$

Answer: The magnitude of the resultant is 30 N.

Example 3: Multi-Step Complete Vector Addition

Problem: Vector A has magnitude 40 m at 30° above the positive x-axis. Vector B has magnitude 50 m at 120° from the positive x-axis. Find (a) the x and y components of the resultant, (b) the magnitude of the resultant, and (c) the direction of the resultant.

This requires multiple steps: resolve both vectors into components, add components, then find magnitude and direction of resultant.

Step 1: Find components of Vector A

$A_x = A \cos(\theta_A) = 40 \cos(30°) = 40 \times 0.866 = 34.6$ m

$A_y = A \sin(\theta_A) = 40 \sin(30°) = 40 \times 0.500 = 20.0$ m

Step 2: Find components of Vector B

$B_x = B \cos(\theta_B) = 50 \cos(120°) = 50 \times (-0.500) = -25.0$ m

$B_y = B \sin(\theta_B) = 50 \sin(120°) = 50 \times 0.866 = 43.3$ m

Part (a): Find resultant components

$R_x = A_x + B_x = 34.6 + (-25.0) = 9.6$ m

$R_y = A_y + B_y = 20.0 + 43.3 = 63.3$ m

Answer (a): $R_x = 9.6$ m and $R_y = 63.3$ m

Part (b): Find magnitude

$R = \sqrt{R_x^2 + R_y^2} = \sqrt{(9.6)^2 + (63.3)^2}$

$R = \sqrt{92.2 + 4007} = \sqrt{4099} = 64.0$ m

Answer (b): The magnitude is 64.0 m.

Part (c): Find direction

$\theta = \tan^{-1}\left(\frac{R_y}{R_x}\right) = \tan^{-1}\left(\frac{63.3}{9.6}\right) = \tan^{-1}(6.59) = 81.4°$

Answer (c): The direction is 81.4° above the positive x-axis.

Summary

  • The resultant vector is the sum of two or more vectors; it has the same effect as all the individual vectors combined.
  • The tip-to-tail method places the tail of the second vector at the tip of the first; the resultant goes from the tail of the first to the tip of the last.
  • The parallelogram method places both vectors tail-to-tail and draws the resultant as the diagonal.
  • Both graphical methods give the same resultant; they're just different visualizations of vector addition.
  • The analytical method uses components: add all x-components to get $R_x$, add all y-components to get $R_y$.
  • Vector addition is commutative: $\mathbf{A} + \mathbf{B} = \mathbf{B} + \mathbf{A}$ (order doesn't matter).
  • After finding $R_x$ and $R_y$, use the Pythagorean theorem to find magnitude: $R = \sqrt{R_x^2 + R_y^2}$.
  • Use inverse tangent to find direction: $\theta = \tan^{-1}(R_y/R_x)$; check the quadrant!
  • The analytical method is more accurate than graphical methods and works for any number of vectors.
  • Vector addition skills are essential throughout physics for analyzing forces, motion, fields, and more.

Practice Problems

1. Explain why you cannot simply add the magnitudes of two vectors to find the resultant. Under what special circumstance would adding magnitudes give the correct answer?

2. Describe the tip-to-tail method for adding vectors graphically. Why is it called "tip-to-tail"?

3. Vector A points east with magnitude 30 m. Vector B points north with magnitude 40 m. Without calculating, estimate whether the resultant will be closer to 50 m, 70 m, or 90 m. Explain your reasoning.

4. If vector A has components $A_x = -15$ m and $A_y = 20$ m, in which quadrant does this vector point? Explain.

5. Vector A has components $A_x = 8$ N and $A_y = 6$ N. Vector B has components $B_x = 12$ N and $B_y = 16$ N. Calculate the x-component of the resultant $\mathbf{R} = \mathbf{A} + \mathbf{B}$.

6. Vector C has components $C_x = 9$ m and $C_y = 12$ m. Vector D has components $D_x = -4$ m and $D_y = -3$ m. Calculate the y-component of the resultant $\mathbf{R} = \mathbf{C} + \mathbf{D}$.

7. A resultant vector has components $R_x = 36$ m/s and $R_y = 48$ m/s. Calculate the magnitude of the resultant.

8. Multi-step problem: Vector F has magnitude 60 N at 37° above the positive x-axis. Vector G has magnitude 80 N at 90° (straight up).
(a) Find the x-component and y-component of vector F. (Use $\cos(37°) = 0.80$ and $\sin(37°) = 0.60$)
(b) Find the x-component and y-component of vector G.
(c) Find the x-component and y-component of the resultant $\mathbf{R} = \mathbf{F} + \mathbf{G}$.
(d) Find the magnitude of the resultant.
(e) Find the direction of the resultant (angle from positive x-axis).

Answer Key

1. You cannot simply add magnitudes because direction matters. If two vectors point in different directions, the resultant depends on both the magnitudes AND the angle between them. For example, 30 N east plus 40 N north gives a 50 N resultant (by Pythagorean theorem), not 70 N. The ONLY time you can add magnitudes directly is when both vectors point in exactly the same direction—then the resultant magnitude equals the sum of the individual magnitudes.

2. The tip-to-tail method involves drawing the first vector, then drawing the second vector starting at the tip (arrowhead/head) of the first vector, maintaining its correct direction and length. The resultant is drawn from the tail of the first vector to the tip of the last vector. It's called "tip-to-tail" because you place the tail of the second vector at the tip of the first vector.

3. The resultant should be close to 50 m. This is a 3-4-5 right triangle scaled by 10 (30-40-50). Using the Pythagorean theorem: $R = \sqrt{30^2 + 40^2} = \sqrt{900 + 1600} = \sqrt{2500} = 50$ m. It won't be 70 m (that would be adding magnitudes), and definitely not 90 m. The resultant of perpendicular vectors is always less than the sum of their magnitudes but more than either individual magnitude.

4. This vector points in the second quadrant (upper left). The x-component is negative (points left) and the y-component is positive (points up). Second quadrant is where x < 0 and y > 0.

5.

Given: $A_x = 8$ N, $A_y = 6$ N, $B_x = 12$ N, $B_y = 16$ N

Find: $R_x$

Formula: $R_x = A_x + B_x$

Substitute: $R_x = 8 + 12$

Solve: $R_x = 20$ N

Answer: The x-component of the resultant is 20 N.

6.

Given: $C_x = 9$ m, $C_y = 12$ m, $D_x = -4$ m, $D_y = -3$ m

Find: $R_y$

Formula: $R_y = C_y + D_y$

Substitute: $R_y = 12 + (-3) = 12 - 3$

Solve: $R_y = 9$ m

Answer: The y-component of the resultant is 9 m.

7.

Given: $R_x = 36$ m/s, $R_y = 48$ m/s

Find: Magnitude $R$

Formula: $R = \sqrt{R_x^2 + R_y^2}$

Substitute: $R = \sqrt{(36)^2 + (48)^2} = \sqrt{1296 + 2304} = \sqrt{3600}$

Solve: $R = 60$ m/s

Answer: The magnitude is 60 m/s.

8.

Part (a): Components of F

$F_x = F \cos(37°) = 60 \times 0.80 = 48$ N

$F_y = F \sin(37°) = 60 \times 0.60 = 36$ N

Answer (a): $F_x = 48$ N, $F_y = 36$ N

Part (b): Components of G

Since G points at 90° (straight up): $\cos(90°) = 0$, $\sin(90°) = 1$

$G_x = 80 \times 0 = 0$ N

$G_y = 80 \times 1 = 80$ N

Answer (b): $G_x = 0$ N, $G_y = 80$ N

Part (c): Resultant components

$R_x = F_x + G_x = 48 + 0 = 48$ N

$R_y = F_y + G_y = 36 + 80 = 116$ N

Answer (c): $R_x = 48$ N, $R_y = 116$ N

Part (d): Magnitude

$R = \sqrt{R_x^2 + R_y^2} = \sqrt{(48)^2 + (116)^2} = \sqrt{2304 + 13456} = \sqrt{15760} = 125.5$ N

Answer (d): The magnitude is 126 N (rounded to 3 sig figs).

Part (e): Direction

$\theta = \tan^{-1}\left(\frac{R_y}{R_x}\right) = \tan^{-1}\left(\frac{116}{48}\right) = \tan^{-1}(2.417) = 67.5°$

Answer (e): The direction is 67.5° above the positive x-axis.
